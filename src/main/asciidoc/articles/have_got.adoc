== Have (got)

*Have (got)* используется:

* чтобы показать, что что-то принадлежит кому-то. *_(He's got a ball)_*
* при описании людей, животных или предметов. *_(She's got blue eyes)_*
* в следующий высказываниях:

====
* I've got a headache
* I've got a temperature
* I've got a cough
* I've got toothache
* I've got a cold
* I've got a problem
====
